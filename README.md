# TripLingo

TripLingo is an iOS app project organized with a feature-first structure for easier scaling and maintenance.

## Project Structure

```text
TripLingo/
├── App/
│   └── TripLingoApp.swift
├── Features/
│   ├── Onboarding/
│   ├── Lessons/
│   ├── Phrasebook/
│   ├── Translate/
│   └── Explore/
├── Models/
├── Services/
├── Resources/
│   └── SeedContent/
└── Core/
    ├── UI/
    ├── Extensions/
    └── Utils/
```

## Getting Started

1. Open `TripLingo.xcodeproj` in Xcode.
2. Select a simulator or device.
3. Build and run the `TripLingo` target.
